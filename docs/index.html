Movie Catalog Project

About the Project

This project is a Java-based movie catalog application. It allows users to manage a collection of movies, including adding, removing, and displaying movie details.

Features

  • Add movies to the catalog
  • Remove movies by title
  • View the catalog through a web interface

How to Run

To run the project, clone the repository and use Maven to build and start the application. Access the web interface at http://localhost:8080.
